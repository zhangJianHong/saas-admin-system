# 模块：配置管理 (doc)

## 变更记录 (Changelog)

- **2025-11-19**: 初始文档创建

## 模块职责

该模块负责 SaaS Admin System 的配置管理和文档维护，包含：
- 系统全局配置文件
- 数据库结构和说明文档

## 文件清单

### 配置文件
- **config.yaml**: 完整的系统配置，包含服务、数据源、国际化等设置
- **db.md**: 数据库文档（当前为空文件）

## 配置详情

### 服务配置
- 端口：8087
- 许可证过期时间：24小时
- 启用调试和监控

### 数据源配置

#### ClickHouse 数据库
- **traces**: 分布式追踪数据 (默认)
- **logs**: 应用日志数据
- **llm**: 大语言模型相关数据
- **rum**: 用户体验监控数据

#### PostgreSQL 数据库
- **light_admin**: 主管理数据库 (默认)
- **llm**: LLM服务专用数据库

#### Redis 缓存
- 默认实例用于会话和缓存管理

### 国际化配置
- 支持语言：中文(zh-CN)、英文(en)
- 缓存启用：一级缓存100条，二级缓存5000条
- 对象池：50个对象，启用预热

### 支付集成
- Stripe 集成配置
- 测试密钥已配置

## 常见问题 (FAQ)

### Q: 配置文件包含明文密码是否安全？
A: 建议使用环境变量或密钥管理系统替换配置文件中的明文密码。

### Q: 如何添加新的数据源？
A: 在 config.yaml 的相应数据源部分添加新的连接配置。

### Q: 国际化配置如何扩展？
A: 在 locale_config.languages 数组中添加新的语言代码，并创建对应的语言文件。
